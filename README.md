# GameDevApp — Level Design Tool

Одностраничное web-приложение для дизайна уровней игр. Все данные хранятся в `localStorage`, никаких зависимостей кроме Material Symbols (Google Fonts).

## Текущий проект: ABERRATION (демо-версия)

9 мастер-сцен + пролог. Каждая сцена содержит события-бусины, каждое событие разбивается на слои:

- **Геймплей** — что игрок делает руками
- **Нарратив** — цель и мотивация
- **История** — диалоги, окружение, лор
- **Эмоция** — что игрок должен чувствовать
- **EYESYS UI** — текст системного интерфейса игры

## Запуск

Открой `level_design.html` в браузере. Всё.

## Управление

- **Перетаскивание холста** — зажми ЛКМ на пустом месте
- **Зум** — колесо мыши
- **Перетаскивание ноды** — за полоску сверху ноды
- **Связи** — тяни от правого порта одной ноды к левому порту другой
- **Выделение группой** — Shift + ЛКМ
- **Undo** — Ctrl+Z
- **Сброс доски** — кнопка «↺ Сбросить доску» в сайдбаре
- **Экспорт/Импорт** — JSON-файл со всем состоянием
